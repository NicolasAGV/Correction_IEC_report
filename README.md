Código para visualizar errores de tipografía entre TRF (.doc documento a ser completado por el técnico), información en el sistema de la empresa y acta de la certificadora. También busca filas cuyo carácter "F" esté incluido e imprime dichas filas para corroborar su correcta utilización.

El mismo interpreta que tipo de TRF y certificadora se utilizaron para localizar de forma correcta los datos a comparar. Finalmente los presenta en un DataFrame (ver foto) para poder ser comparado por el usuario final junto a las filas con “F”. Por la naturaleza de algunos valores, no se realiza una comparación True / False caracter a caracter ya que aunque los datos sufran pequeñas modificaciones, pueden ser equivalentes.
